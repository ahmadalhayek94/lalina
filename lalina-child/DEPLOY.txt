LALINA CHILD THEME — DEPLOYMENT GUIDE
======================================
"Where Craft Meets Confidence."

HOW TO INSTALL
--------------
1. Zip the entire "lalina-child" folder
2. Go to: WordPress Admin → Appearance → Themes → Add New → Upload Theme
3. Upload the zip and click Install Now
4. Click Activate
5. The theme is live immediately

AFTER ACTIVATION — MANUAL STEPS
---------------------------------
1. BLOG LINK FIX
   WP Admin → Appearance → Menus → Footer Menu
   Find the "Blogs" link → change URL to your blog page URL

2. FLATSOME REGISTRATION
   WP Admin → Flatsome → Theme Registration
   Enter your real Envato purchase code

3. REMOVE BLANKET 20% DISCOUNT
   WP Admin → WooCommerce → Coupons
   Find any always-active coupon → deactivate or set an expiry date

4. KLAVIYO EMAIL
   Install Klaviyo plugin → enter API keys

5. GA4 + META PIXEL
   Install Google Site Kit plugin

6. CURRENCY SWITCHER
   Install WOOCS (WooCommerce Currency Switcher) plugin
